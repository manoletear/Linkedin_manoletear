# Preview Your Video

> Source: https://www.remotion.dev/docs/preview

Preview your Remotion video by starting the [Remotion Studio](./studio.md).

## Starting the Preview

```bash
# Regular templates
npm run dev

# Next.js and React Router 7 templates
npm run remotion

# Direct CLI command
npx remotion studio
```

A server starts on port `3000` (or higher if unavailable) and the Remotion Studio opens in the browser.

> Note: In older projects, `npm start` was used instead of `npm run dev`.

## Remotion Studio Features

The Studio provides:
- Real-time video preview with timeline scrubbing
- Frame-by-frame navigation
- Props editing via the sidebar
- Composition selection
- Rendering options
