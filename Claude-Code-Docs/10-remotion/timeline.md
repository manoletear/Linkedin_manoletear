# Remotion Timeline

> Source: https://www.remotion.dev/docs/timeline

A paid customizable timeline component that integrates seamlessly with [Remotion Player](./player.md).
Perfect for builders who need a layer-based editing interface for their video editing app.

Purchase at: https://www.remotion.pro/timeline

## Key Features

**Interface:**
- Multiple track support with drag-and-drop functionality (between and within tracks)
- Timeline item selection and manipulation
- Keyboard shortcuts for common actions
- `CanvasComposition` for rendering timeline state in the Remotion Player
- Zoomable timeline and `<ZoomSlider>` component to control zoom level

**Customization:**
- Theming with variables (Tailwind CSS V4 and V3)
- Extensible state management
